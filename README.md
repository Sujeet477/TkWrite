# TkWrite

A simple and beginner-friendly **TkWrite** built using **Python and Tkinter**.  
This project allows users to create, open, edit, and save text files using a clean graphical interface.

---

##  Features

- Create a new text file  
- Open existing `.txt` files  
- Save text files easily  
- Simple and user-friendly GUI  
- Uses Python’s built-in Tkinter library  

---

##  Technologies Used

- Python  
- Tkinter (GUI Library)
